## Header
```
# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag
```
## Bold
```
**Contoh**
__Contoh__.
```
## Italic
```
*Contoh*
_Contoh_.
```
## Blockquote
Untuk menambahkan <code>blockquote</code> tambahkan <code>></code> sebelum/pada awal paragraf
```
> Tes penulisan blockquote
```
Hasilnya
> Tes penulisan blockquote
